# Ролевая траектория созидателя

## 0. Назначение

Этот документ описывает **ролевую траекторию развития созидателя** — последовательность ключевых ролей от начинающего Ученика до публичного Просветителя.

**FPF A.7 Strict Distinction**: Роль ≠ Метод ≠ Работа. Каждая роль имеет связанные методы и работы.

---

## 1. Пять ключевых ролей

```
Ученик → Интеллектуал → Профессионал → Исследователь → Просветитель
   ↓           ↓              ↓               ↓              ↓
 Личное    Понятийная     Прикладное     Исследоват.    Публичное
развитие    работа        мастерство      мастерство    влияние
```

| ID | Роль | Основная функция | Ключевое мастерство |
|----|------|------------------|---------------------|
| [PD.ROLE.001](PD.ROLE.001-learner.md) | **Ученик** | Освоение нового, изменение себя | Практики саморазвития |
| [PD.ROLE.002](PD.ROLE.002-intellectual.md) | **Интеллектуал** | Мышление, работа с понятиями | Мыслительное мастерство |
| [PD.ROLE.003](PD.ROLE.003-professional.md) | **Профессионал** | Создание рабочих продуктов | Прикладное мастерство |

## 2. Логика перехода между ролями

### 2.1. Последовательность освоения

Роли осваиваются последовательно, но не исключают параллельного проявления:

1. **Ученик** — базовый уровень, необходимый для всех
2. **Интеллектуал** — надстройка над учеником
3. **Профессионал** — применение интеллектуальных навыков
4. **Исследователь** — углубление за пределы известного
5. **Просветитель** — масштабирование влияния

### 2.2. Ворота между ролями

Переход между ролями требует:
1. Накопления артефактов предыдущей роли
2. Демонстрации компетенций
3. Признания сообществом/наставниками
4. Запроса на следующий уровень

---

## 3. Связь ролей с мастерством

| Роль | Тип мастерства | Ключевые методы |
|------|----------------|-----------------|
| Ученик | Практики саморазвития | Слоты, экзокортекс, мировоззрение |
| Интеллектуал | Мыслительное мастерство | Картирование смыслов, тезисирование |
| Профессионал | Прикладное мастерство | Декомпозиция, планирование, требования |
| Исследователь | Исследовательское мастерство | Гипотезы, эксперименты, анализ |
| Просветитель | Просветительское мастерство | Подготовка материалов, фасилитация |

---

## 4. Типичные ловушки (Failure Modes)

| Ловушка | Описание | Связь с ролью |
|---------|----------|---------------|
| **Застревание в Ученике** | Бесконечное обучение без продуктов | Ученик → Интеллектуал |
| **Пропуск уровней** | Попытка стать Просветителем без базы | Любой переход |
| **Отказ от публичности** | «Дело говорит само за себя» | Профессионал → Просветитель |
| **Публичность без дела** | Инфобизнес без экспертизы | Просветитель |
| **Одиночка-эксперт** | Отказ от передачи знаний | Исследователь → Просветитель |

# Роль: Исследователь (Researcher)

## 1. Определение

**Исследователь** — четвёртая роль в траектории созидателя. Расширяет фронтир знаний, создаёт новые методы.

**Суть роли:**
- Расширяет фронтир знаний
- Создаёт новые методы и подходы
- Проводит эксперименты и проверяет гипотезы
- Вносит вклад в развитие дисциплины

## 2. Ключевое мастерство

**Исследовательское мастерство:**
- Новизна — работа на границе известного
- Критический анализ существующих подходов
- Эксперименты и проверка гипотез
- Вклад в развитие дисциплины

## 4. Артефакты роли

- Исследовательские отчёты
- Новые методы и практики
- Экспериментальные проекты
- Научные публикации

## 5. Критерии перехода к Просветителю

| Критерий | Описание |
|----------|----------|
| **Вклад** | Создан значимый вклад — метод, теория, инструмент |
| **Признание** | Признание в профессиональном сообществе |
| **Запрос** | Запрос на передачу знаний широкой аудитории |
| **Готовность** | Готовность к публичной деятельности |

## 6. Вопросы для самооценки

- Создал ли я что-то новое?
- Признаёт ли это сообщество?
- Публикуюсь ли я?
- Есть ли эксперименты?

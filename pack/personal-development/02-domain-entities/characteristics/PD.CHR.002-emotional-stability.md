# Характеристика: Эмоциональная стабильность (Emotional Stability)

## 1. Определение

Эмоциональная стабильность -- способность агента возвращаться в равновесное состояние после эмоционального сбоя (стресс, неудача, конфликт, разочарование). Не отсутствие эмоциональных реакций, а скорость и надёжность восстановления.

## 2. Модель неваляшки (Tumbler Model)

```
Равновесие      Сбой         Восстановление     Равновесие
    O          O              O                    O
   /|\   -->   \   -->       /|   -->             /|\
   / \         /\            / \                  / \
   ===         ===           ===                  ===

Скорость восстановления = показатель эмоциональной стабильности
```

| Аспект | Неваляшка | Агент |
|--------|-----------|-------|
| Сбой | Толчок -- отклонение от вертикали | Стресс, неудача, конфликт -- отклонение от рабочего состояния |
| Восстановление | Центр тяжести возвращает в вертикаль | Практики + привычки возвращают в равновесие |
| Не означает | Невозможность опрокинуть | Невозможность расстроиться |
| Означает | Надёжное возвращение | Надёжное возвращение в рабочее состояние |

Ключевой инсайт: стабильность -- не жёсткость (ригидность), а упругость (resilience). Жёсткий объект при сильном ударе ломается; неваляшка -- возвращается.

## 3. CSLC-описание (FPF A.18)

| Компонент | Значение |
|-----------|----------|
| **Characteristic** | Эмоциональная стабильность |
| **Scale** | Ordinal (порядковая) |
| **Levels** | 1-5 |
| **Coordinates** | 1: «Любой сбой выводит из строя на часы/дни» -- 5: «Восстанавливается за минуты после серьёзного сбоя» |

| Уровень | Описание | Время восстановления |
|---------|----------|---------------------|
| 1 | Любой стресс парализует; долгое «залипание» в эмоциях | Дни |
| 2 | Восстанавливается, но медленно; может потерять целый день | Часы -- день |
| 3 | Замечает сбой, возвращается через осознанное усилие | Часы |
| 4 | Быстро корректирует; использует стоп-моменты и методы | Минуты -- час |
| 5 | Восстановление почти автоматическое; сбой не нарушает планов | Минуты |

## 4. Методы развития

| Метод | Как влияет на стабильность |
|-------|---------------------------|
| **Собранность** (PD.FORM.024) | Пять действий собранного человека создают структуру, снижающую амплитуду сбоев |
| **Мышление письмом (TIW)** | Перевод эмоций в текст снижает их интенсивность; осознание причин |
| **Формирование привычек** | Автоматизация рутин снижает когнитивную нагрузку; меньше точек уязвимости |
| **Стоп-моменты** (PD.STATE.003) | Раннее обнаружение сбоя = меньшая амплитуда отклонения |
| **Ролевое разделение** (PD.FORM.031) | Эмоции в одной роли не переносятся на другую |

## 5. Место в иерархии характеристик

Эмоциональная стабильность -- 4-я по приоритету характеристика (PD.FORM.029):
1. Агентность (без неё не начать)
2. Мастерство (без него действия неэффективны)
3. Ресурсность (без неё мастерство не реализуется)
4. **Эмоциональная стабильность** (без неё результаты нестабильны)
5. Баланс (без него успех не интегральный)

Стабильность обеспечивает устойчивость всей системы: без неё достижения в агентности, мастерстве и ресурсности могут быть обнулены одним серьёзным сбоем.

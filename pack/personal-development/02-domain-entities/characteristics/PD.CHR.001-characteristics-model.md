# Модель характеристик созидателя

## 1. Концепция

Модель характеристик устанавливает связь:

```
Характеристика → Показатель → Значение → Состояние
     (CHR)         (Metric)    (Value)    (State)
```

## 2. Определения (FPF A.7 Strict Distinction)

| Термин | FPF-тип | Определение |
|--------|---------|-------------|
| **Характеристика** | U.Characteristic | Измеримое свойство системы (созидателя) |
| **Показатель** | U.Measure | Конкретная метрика для измерения характеристики |
| **Значение** | U.Value | Текущее значение показателя |
| **Состояние** | U.State | Совокупность значений всех показателей в данный момент |

## 3. CSLC-стандарт (FPF A.18)

Каждая характеристика описывается по стандарту **CSLC**:

| Компонент | Описание | Пример |
|-----------|----------|--------|
| **C**haracteristic | Название характеристики | Агентность |
| **S**cale | Тип шкалы измерения | ordinal (порядковая) |
| **L**evels | Уровни/градации | 1-5 |
| **C**oordinates | Якоря шкалы | "Реактивный" → "Проактивный" |

## 5. CharacteristicSpace (FPF A.19)

Пространство характеристик созидателя — многомерное пространство, где:
- Каждая характеристика — отдельная ось
- Состояние созидателя — точка в этом пространстве
- Развитие — траектория движения точки

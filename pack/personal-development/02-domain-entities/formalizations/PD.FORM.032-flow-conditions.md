# Формализация: Условия потока (Flow Conditions)

## 1. Определение

Поток (flow) -- оптимальное состояние деятельности, описанное Чиксентмихайи (Csikszentmihalyi), в котором агент полностью погружён в задачу, теряет ощущение времени и показывает максимальную производительность. Три группы условий потока могут быть осознанно созданы через практики ролевого мастерства (PD.FORM.031).

## 2. Три группы условий

### 2.1. Баланс сложности и мастерства (Complexity-Mastery Match)

| Аспект | Описание |
|--------|----------|
| **Условие** | Сложность задачи приблизительно равна уровню мастерства агента |
| **Слишком легко** | Скука, потеря интереса, автопилот |
| **Слишком сложно** | Тревога, фрустрация, ступор |
| **Управление через ролевое мастерство** | Идентификация роли: выбрать задачу, соответствующую текущему уровню; декомпозировать сложную задачу |

```
Тревога     |         /
            |       / ПОТОК
            |     /
Скука       |   /
            | /
            +------------>
            Мастерство
```

### 2.2. Концентрация в моменте (In-the-Moment Concentration)

| Аспект | Описание |
|--------|----------|
| **Условие** | Полное внимание направлено на текущее действие; осознанная (S2) работа с концептуальным вниманием |
| **Препятствия** | Шесть типов потери внимания (PD.FORM.025) |
| **Управление через ролевое мастерство** | Самоконтроль: стоп-моменты, управление средой, ритуал входа, отключение отвлечений |

Концентрация в потоке -- не волевое усилие, а результат правильно настроенных условий: интересная задача нужной сложности + защищённая среда + концептуальный (а не эмоциональный) фокус.

### 2.3. Обратная связь после действия (Post-Action Feedback)

| Аспект | Описание |
|--------|----------|
| **Условие** | Агент получает ясную обратную связь о результатах своих действий |
| **Без обратной связи** | Невозможно оценить прогресс; теряется ощущение смысла |
| **Управление через ролевое мастерство** | Вход/выход из роли: фиксация результата при завершении; ритуал закрытия; сравнение с планом |

## 3. Поток и собранность

| Собранность (PD.FORM.024) | Поток |
|---------------------------|-------|
| Осознанные действия по управлению вниманием | Результат: состояние полного погружения |
| Усилие по удержанию фокуса | Фокус удерживается «сам» (условия созданы) |
| Многомасштабное внимание | Внимание сужено до одного масштаба (текущий момент) |

Собранность -- предусловие потока. Собранный человек создаёт условия, в которых поток возникает естественно.

## 4. Практические рекомендации

| Группа условий | Что проверить перед работой |
|----------------|---------------------------|
| Баланс сложности | Задача не слишком проста и не слишком сложна? Могу ли декомпозировать? |
| Концентрация | Среда защищена? Телефон убран? Ритуал входа выполнен? |
| Обратная связь | Как я узнаю, что продвинулся? Есть ли критерий завершения? |

# Формализация: Действия собранного человека (Composure Actions)

## 1. Определение

Собранность (composure) -- не внутреннее состояние, а набор наблюдаемых действий. Человек собран тогда, когда выполняет пять конкретных поведений. Собранность определяется через действия, а не через ощущения.

## 2. Пять действий собранного человека

| # | Действие | Описание | Наблюдаемый признак |
|---|----------|----------|---------------------|
| 1 | **Различает важное от текущего/срочного** | Не путает срочность с важностью; принимает решения на основе приоритетов, а не давления | Выбирает задачу по стратегическому весу, а не по горящему дедлайну |
| 2 | **Работает рационально** | Использует защищённое время, ритуалы входа в работу, структурированные методы | Есть расписание, ритуал, рабочее место; не работает хаотично |
| 3 | **Удерживает несколько временных масштабов одновременно** | Видит текущую задачу в контексте проекта, недели, стратегии | В любой момент может ответить: что делаю, зачем, как это связано с целью |
| 4 | **Замечает потерю внимания и корректирует** | Отслеживает моменты отвлечения и возвращает фокус | Использует стоп-моменты; замечает, когда ушёл в телефон или размышления |
| 5 | **Направляет внимание с помощью понятий, а не эмоций** | Использует концепции (роль, метод, приоритет) для управления фокусом, а не реагирует на эмоциональные импульсы | Описывает свои действия через роли и методы, а не через «хочу/не хочу» |

## 3. Собранность как практика, а не свойство

```
Собранность != черта характера
Собранность  = повторяемый набор действий
                  |
                  v
            Может быть установлена через практику
            Может быть потеряна при прекращении практики
```

- Собранность не бывает «от природы» -- она результат осознанной тренировки
- Потеря собранности -- нормальное явление; ключевой навык -- восстановление

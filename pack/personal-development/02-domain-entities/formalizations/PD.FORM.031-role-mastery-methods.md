# Формализация: Методы ролевого мастерства (Role Mastery Methods)

## 1. Определение

Ролевое мастерство -- способность агента осознанно входить в роли, действовать в них эффективно и выходить без потери контроля. Реализуется через три метода: идентификация ролей, вход/выход из роли и самоконтроль. Ролевое мастерство -- условие потока (PD.FORM.032) и собранности (PD.FORM.024).

## 2. Три метода

### 2.1. Идентификация ролей (Role Identification)

| Аспект | Описание |
|--------|----------|
| **Суть** | Обнаружение ролей через деятельностный кругозор: какие роли нужны для реализации моих проектов и жизненных сфер? |
| **Вход** | Список проектов, сфер жизни, неудовлетворенностей |
| **Выход** | Список ролей с описанием: что делает, какой рабочий продукт, какое мастерство нужно |
| **Метод** | Анализ деятельности: «Для этого проекта мне нужна роль X, потому что я произвожу рабочий продукт Y» |
| **Частота** | При стратегировании (еженедельно / ежемесячно) |

### 2.2. Вход/выход из роли (Role Entry / Exit)

| Аспект | Описание |
|--------|----------|
| **Суть** | Осознанный переход между ролями через ритуалы и стоп-моменты |
| **Вход в роль** | Ритуал: смена контекста (рабочее место, инструмент, документ), объявление роли, настройка внимания |
| **Выход из роли** | Стоп-момент (PD.STATE.003): фиксация результата, очистка контекста, переключение |
| **Опасность** | Смешение ролей: «ответил на рабочее письмо, находясь в роли родителя» |
| **Критерий** | Агент может назвать свою текущую роль в любой момент |

### 2.3. Самоконтроль (Self-Control)

| Аспект | Описание |
|--------|----------|
| **Суть** | Управление вниманием и эмоциональное разделение между ролью и личностью |
| **Внимание** | Направление фокуса на объекты роли (а не на отвлечения); использование типов потери внимания (PD.FORM.025) |
| **Эмоциональное разделение** | Эмоции роли не переносятся на другие роли; неудача в роли != неудача личности |
| **Инструмент** | Стоп-моменты, мышление письмом, различение «я в роли» vs «я как личность» |

## 3. Связь между методами

```
Идентификация ролей       Вход/выход из роли        Самоконтроль
(какие роли мне нужны?)   (когда и как переключаюсь?)  (удерживаю ли фокус?)
         |                         |                         |
         v                         v                         v
Список ролей     -->     Ритуалы + стоп-моменты  -->  Устойчивый фокус
                                                           |
                                                           v
                                                    Условия потока
                                                    (PD.FORM.032)
```

## 4. Ролевое мастерство как условие потока

Три группы условий потока (PD.FORM.032) управляемы через ролевое мастерство:
1. **Сложность ~ мастерство** -- через идентификацию: выбор роли, соответствующей уровню
2. **Концентрация** -- через самоконтроль: управление вниманием
3. **Обратная связь** -- через вход/выход: фиксация результатов, ритуал закрытия

# Формализация: Классификация досуговых практик (Leisure Practice Classification)

## 1. Определение

**Классификация досуговых практик** — формализация пяти классов досуга для осознанной организации отдыха. Классификация помогает обеспечить разнообразие отдыха и избежать перекоса в сторону пассивных форм.

## 2. Пять классов

| # | Класс | Описание | Примеры |
|---|-------|----------|---------|
| 1 | **Двигательно-спортивный** (Motor/Sports) | Физическая активность, движение | Спорт, танцы, йога, плавание |
| 2 | **Поддержание здоровья** (Health Maintenance) | Профилактика и восстановление | Диагностика, санаторий, массаж |
| 3 | **Культурно-познавательный** (Cultural-Educational) | Расширение кругозора через впечатления | Театр, выставки, путешествия, концерты |
| 4 | **Пассивный отдых** (Passive Rest) | Восстановление через минимум активности | Прогулки, сон, соцсети, кино |
| 5 | **"Хотя бы раз в жизни"** (At Least Once) | Реализация мечт и уникальных желаний | Прыжок с парашютом, кругосветка, северное сияние |

## 3. Правила классификации

- Личная классификация приветствуется — можно добавлять подклассы
- Задача — обеспечить **разнообразие**: не весь досуг должен быть в одном классе
- Класс 5 ("Хотя бы раз") планируется стратегически, на горизонте года

## 4. Связь с планированием

| Класс | Частота планирования |
|-------|---------------------|
| 1 (Спорт) | Еженедельно |
| 2 (Здоровье) | Ежеквартально / по расписанию |
| 3 (Культура) | Ежемесячно |
| 4 (Пассивный) | Ежедневно (не требует планирования) |
| 5 (Мечты) | Ежегодно |

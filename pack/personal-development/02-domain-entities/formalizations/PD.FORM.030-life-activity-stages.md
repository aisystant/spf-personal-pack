# Формализация: Стадии жизнедеятельности (Life Activity Stages)

## 1. Определение

Стадии жизнедеятельности -- шесть универсальных видов деятельности созидателя, образующих циклический (не последовательный) процесс. В любой момент времени агент находится в одной из стадий. Полноценная жизнедеятельность требует регулярного прохождения всех шести стадий.

## 2. Шесть стадий

| # | Стадия | Суть | Основной метод | Результат |
|---|--------|------|---------------|-----------|
| 1 | **Потребление информации** (information consumption) | Получение новой информации из внешних источников | SSR (Study, Synthesis, Reflection) | Новые факты, знания, впечатления |
| 2 | **Рефлексия / мышление** (reflection / thinking) | Переработка информации и опыта в знания и решения | TIW (Thinking-in-Writing), TbS (Thinking-by-Speaking) | Осмысленный опыт, гипотезы, выводы |
| 3 | **Стратегирование** (strategizing) | Определение направления: что делать и зачем | PD.METHOD.008 Strategizing | Стратегия, приоритеты, цели |
| 4 | **Планирование** (planning) | Определение порядка: когда, как и сколько | PD.METHOD.009 Planning | План, расписание, бюджет времени |
| 5 | **Исполнение** (execution) | Реализация планов через прикладное мастерство | Ролевое мастерство (PD.FORM.031) | Рабочие продукты, результаты |
| 6 | **Досуг** (leisure) | Восстановление и развлечение | PD.METHOD.006 Leisure | Восстановленная энергия, впечатления |

## 3. Циклическая природа

```
    Потребление  -->  Рефлексия  -->  Стратегирование
         ^                                  |
         |                                  v
      Досуг  <--  Исполнение  <--  Планирование
```

Стадии циклические, а не последовательные:
- Можно начать с любой стадии
- Возможны петли (рефлексия -> потребление -> рефлексия)
- Невозможно «завершить цикл» -- он повторяется всю жизнь
- Здоровый ритм: все шесть стадий проходятся регулярно (еженедельно)

## 4. Дисбалансы

| Пропущенная стадия | Последствие |
|--------------------|-------------|
| Потребление | Интеллектуальная стагнация; решения на устаревших знаниях |
| Рефлексия | Накопление необработанного опыта; повторение ошибок |
| Стратегирование | Много действий без направления; «белка в колесе» |
| Планирование | Хорошие идеи без реализации; хаотичная работа |
| Исполнение | Всё продумано, но ничего не сделано; прокрастинация |
| Досуг | Выгорание; снижение качества всех остальных стадий |

## 5. Связь с четырьмя базовыми активностями

Стадии жизнедеятельности -- расширение PD.FORM.008 (Four Basic Activities):
- Потребление + Рефлексия = «Познание»
- Стратегирование + Планирование = «Управление»
- Исполнение = «Созидание»
- Досуг = «Восстановление»

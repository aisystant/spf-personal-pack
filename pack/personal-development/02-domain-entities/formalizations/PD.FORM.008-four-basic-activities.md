# Формализация: Четыре базовых вида деятельности (Four Basic Human Activities)

## 1. Определение

**Четыре базовых вида деятельности** — классификация человеческой активности для подбора практик саморазвития. Практики были отобраны так, чтобы покрыть все четыре вида деятельности.

## 2. Четыре вида

| # | Вид деятельности | Описание | Примеры практик |
|---|------------------|----------|-----------------|
| 1 | **Потребление информации** (Information Consumption) | Приём и обработка входящей информации | Чтение (SSR), формирование окружения |
| 2 | **Рефлексия (познание)** (Reflection / Cognition) | Осмысление, переработка информации в знание | Мышление письмом (TIW), мышление проговариванием (TbS), стратегирование, планирование |
| 3 | **Действие** (Action) | Целенаправленная активность, изменяющая мир | Планирование, инвестирование времени, формирование окружения |
| 4 | **Отдых** (Rest) | Восстановление ресурсов | Организация досуга |

## 4. Правило полноты

> Набор практик саморазвития должен покрывать все 4 вида деятельности. Перекос в одну сторону (например, только чтение без рефлексии) снижает эффективность развития.

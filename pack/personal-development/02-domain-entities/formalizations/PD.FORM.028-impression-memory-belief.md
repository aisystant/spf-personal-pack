# Формализация: Впечатления -- Воспоминания -- Убеждения (Impression -- Memory -- Belief)

## 1. Определение

Обработка опыта -- трёхстадийный процесс, в котором внешний опыт трансформируется во внутренние конструкты, определяющие поведение агента. Каждая стадия является точкой вмешательства для саморазвития: на впечатления можно влиять через управление средой, на воспоминания -- через рефлексию, на убеждения -- через осознанную замену.

## 2. Три стадии

| Стадия | Определение | Характеристики | Пример |
|--------|-------------|----------------|--------|
| **Впечатления** (impressions) | Эмоционально окрашенные следы опыта | Свежие, яркие, нестабильные; подвержены искажению | Ощущение от первого дня на новой работе |
| **Воспоминания** (memories) | Сохранённые эпизоды, отфильтрованные существующими убеждениями | Избирательные; подтверждают существующую картину мира; реконструируются при извлечении | «Помню, что было трудно» (деталь потеряна, осталась оценка) |
| **Убеждения** (beliefs) | Стабильные конструкты для принятия решений | Устойчивые; имеют наивысший вес достоверности для агента; меняются медленно | «Я не способен к математике» |

## 3. Процесс трансформации

```
Опыт (событие)
    |
    v
Впечатления (эмоционально окрашенные следы)
    |  фильтр: существующие убеждения
    v
Воспоминания (сохранённые, но искажённые эпизоды)
    |  накопление + генерализация
    v
Убеждения (стабильные конструкты для решений)
    |
    v
Влияют на фильтрацию новых впечатлений --> цикл
```

Цикл является самоподдерживающимся: убеждения фильтруют впечатления, которые формируют воспоминания, которые укрепляют убеждения.

## 4. Точки вмешательства для саморазвития

| Стадия | Метод вмешательства | Описание |
|--------|---------------------|----------|
| **Впечатления** | Управление средой и опытом | Осознанно выбирать, какой опыт получать (книги, окружение, проекты) |
| **Воспоминания** | Рефлексия и мышление письмом (TIW) | Перерабатывать воспоминания через осознанный анализ; выявлять искажения |
| **Убеждения** | Замена ложных убеждений на SoTA-знания | Обнаруживать убеждения с плохой предсказательной силой; заменять на проверенные модели |

## 5. Связь с мировоззрением

Убеждения -- один из трёх слоёв мировоззрения (PD.FORM.022):
- **Факты** -- сырая информация
- **Знания** -- применимая информация на основе теорий
- **Убеждения** -- конструкты из обработанного опыта

Цель развития: на месте ложных убеждений поставить проверенные знания (SoTA-трансдисциплины).

# Формализация: Степени мастерства (Mastery Degrees)

## 1. Определение

**Степени мастерства** — квалификационная формализация, описывающая четыре уровня освоения любой практики. Каждая степень характеризуется качеством рабочего продукта, уровнем автоматизма и внутренней мотивацией.

## 2. Четыре степени

### Степень 1: Объяснение (Explanation)

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Может объяснить зачем, что, как; оперирует понятиями |
| **Качество РП** | Не производит самостоятельно |
| **Мотивация** | "Любопытно" или "надо" |
| **Автоматизм** | Отсутствует |

### Степень 2: Умение (Skill)

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Систематически применяет, удовлетворительное качество РП |
| **Качество РП** | Удовлетворительное |
| **Мотивация** | "Я могу" |
| **Автоматизм** | Низкий, требует осознанного усилия |

### Степень 3: Навык (Habit)

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Применяет автоматически, фокус на качестве, получает удовольствие от практики |
| **Качество РП** | Хорошее |
| **Мотивация** | "Я хочу" |
| **Автоматизм** | Высокий, действие не требует волевого усилия |

### Степень 4: Мастерство (Mastery)

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Работает в неидеальных условиях, учит других, развивает теорию |
| **Качество РП** | Отличное, в том числе в нештатных ситуациях |
| **Мотивация** | "Не могу НЕ делать" |
| **Автоматизм** | Полный, адаптивный |

## 3. Критерии перехода

```
Объяснение → Умение
    │ Первый самостоятельный РП удовлетворительного качества
    ↓
Умение → Навык
    │ Автоматизм + удовольствие от практики
    ↓
Навык → Мастерство
    │ Работа в неидеальных условиях + обучение других
```

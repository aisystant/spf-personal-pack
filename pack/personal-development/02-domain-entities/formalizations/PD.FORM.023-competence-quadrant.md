# Формализация: Квадрант компетенций (Competence Quadrant)

## 1. Определение

Квадрант компетенций -- модель 2x2, описывающая четыре состояния освоения навыка через пересечение двух осей: осознанность (conscious / unconscious) и компетентность (competent / incompetent). Модель фиксирует не уровень мастерства, а отношение агента к собственному уровню.

## 2. Четыре состояния

| Состояние | Осознанность | Компетентность | Описание |
|-----------|-------------|----------------|----------|
| **Неосознанная некомпетентность** | Нет | Нет | Агент не знает, что не умеет. Самое опасное состояние: нет сигнала для начала обучения |
| **Осознанная некомпетентность** | Да | Нет | Агент понимает, что не умеет. Точка начала обучения: появляется неудовлетворенность и мотивация |
| **Осознанная компетентность** | Да | Да | Агент умеет, но требуется усилие и концентрация. Навык выполняется через сознательный контроль |
| **Неосознанная компетентность** | Нет | Да | Навык автоматизирован до уровня привычки. Мастерство: выполнение без осознанного усилия |

## 3. Переходы между состояниями

```
Неосознанная           Осознанная            Осознанная            Неосознанная
некомпетентность  -->  некомпетентность  -->  компетентность  -->  компетентность
   (не знаю,             (знаю, что            (умею, но             (умею
    что не умею)          не умею)               с усилием)           автоматически)
       |                     |                     |                     |
    Обратная              Обучение              Практика              Привычка
    связь / шок           начинается            + повторение          сформирована
```

## 4. Опасность неосознанной некомпетентности

- Агент не ищет обучения, потому что не видит дефицита
- Принимает решения на основе ложной уверенности
- Единственный выход -- внешняя обратная связь или столкновение с реальностью (провал)
- Связано с эффектом Даннинга-Крюгера: низкая компетентность + высокая самооценка

## 5. Практическое применение

| Контекст | Как использовать |
|----------|-----------------|
| **Самодиагностика** | Определить, в каком квадранте находится навык; искать слепые зоны |
| **Наставничество** | Помочь ученику перейти из неосознанной в осознанную некомпетентность (самый важный шаг) |
| **Планирование обучения** | Разные квадранты требуют разных методов: обратная связь, теория, практика, автоматизация |

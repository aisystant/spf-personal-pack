# Формализация: Модель агента (Agent Model — Personality + Organism)

## 1. Определение

Агент как система рассматривается с двух точек зрения: функциональной (что делает) и модульной (из чего состоит). Обе декомпозиции необходимы для понимания саморазвития: функциональная — для определения направлений развития, модульная — для понимания средств.

## 2. Функциональная декомпозиция

| Подсистема | Аналогия | Что входит | Функция |
|-----------|----------|-----------|---------|
| **Организм** | Hardware (аппаратная часть) | Тело, мозг, экзотело (деньги, имущество) | Аппаратные функции: зрение, слух, говорение, передвижение |
| **Личность** | Software (программная часть) | Совокупность всех способностей (мастерств) | Реализует желания, жизненное мастерство, эмоции |

Личность в свою очередь:

| Компонента личности | Содержание |
|--------------------|-----------|
| **Мыслительное мастерство** (интеллект) | Трансдисциплины, способность рассуждать, строить модели |
| **Прикладное мастерство** | Профессиональные методы, навыки конкретных ролей |
| **Эмоциональная часть** | Впечатления, убеждения, ценности; механизм перепрограммирования через опыт |

## 3. Модульная декомпозиция

| Модуль | Hardware | Software |
|--------|----------|----------|
| **Биологическое тело** (включая мозг) | Организм как «овощ» (зрение, слух, моторика) | Личность: мастерства, мировоззрение, убеждения |
| **Экзотело** | Экзоорганизм: одежда, деньги, инструменты, автомобиль | Экзокортекс: компьютер, смартфон, приложения, AI-агенты |

## 4. Роль экзокортекса

- Вычислительная мощность биологического мозга не изменилась за тысячи лет
- За счёт экзокортекса современная личность имеет на порядок более высокие вычислительные мощности
- Не все люди умеют использовать экзокортекс — необходимо осваивать практики саморазвития
- Современный человек отличается от человека прошлого именно за счёт экзокортекса

## 5. Впечатления как механизм обучения

При действии в роли агент получает два результата:
1. **Рабочий продукт** — изменение в мире
2. **Впечатление** — след, впечатывающийся в мозг; изменяет мировоззрение, формирует убеждения, может дать новое мастерство при успехе

## 6. Следствия для саморазвития

Для полноценного саморазвития необходимо уделять внимание всем четырём квадрантам:
1. Жизненное мастерство (софт биологического тела)
2. Эмоциональная часть личности (коррекция мировоззрения)
3. Организм (здоровье, сон, ЗОЖ — хард биологического тела)
4. Экзотело (экзокортекс, инструменты, ресурсы)

# Формализация: Масштабы внимания (Attention Time Scales)

## 1. Определение

Масштабы внимания -- три одновременных горизонта, в которых собранный человек удерживает фокус. Не последовательные этапы, а параллельные слои: агент в любой момент времени может ответить на вопрос «что я делаю?» на каждом из трёх масштабов.

## 2. Три масштаба

| Масштаб | Горизонт | Объект внимания | Характерный вопрос | Инструмент |
|---------|----------|----------------|--------------------|-----------|
| **Краткосрочный** | Минуты -- часы | Текущая задача, помодоро, операция | «Что я делаю прямо сейчас?» | Таймер, чеклист, ритуал входа |
| **Среднесрочный** | Дни -- недели | Проект, рабочий продукт, план недели | «К какому результату я иду на этой неделе?» | WeekPlan, DayPlan, контекст РП |
| **Долгосрочный** | Месяцы -- годы | Стратегия, жизненные цели, миссия | «Зачем я это делаю в масштабе жизни?» | Стратегирование, Strategy.md |

## 3. Одновременное удержание

```
Долгосрочный:   [========= Миссия / Стратегия ==========]
                          |
Среднесрочный:        [== Проект / Неделя ==]
                              |
Краткосрочный:            [Задача]
                              |
                         Текущий момент
```

Собранный человек (PD.FORM.024, действие #3) удерживает все три масштаба одновременно:
- Выполняет текущую задачу (краткосрочный)
- Понимает, как она связана с проектом (среднесрочный)
- Понимает, зачем этот проект в контексте стратегии (долгосрочный)

## 4. Потеря масштаба

| Потерянный масштаб | Симптом | Последствие |
|-------------------|---------|-------------|
| Краткосрочный | Не может сфокусироваться на задаче | Низкая продуктивность, прокрастинация |
| Среднесрочный | Делает задачи, не связанные с проектом | Много действий, мало результата |
| Долгосрочный | Проекты не связаны со стратегией | Достижения не приближают к целям |
| Все три | Разобранность (PD.STATE.002) | Хаотичная деятельность без направления |

Потеря масштаба -- один из шести типов потери внимания (PD.FORM.025, тип #4).

## 5. Практика удержания

Регулярные стоп-моменты (PD.STATE.003) на каждом масштабе:
- **Краткосрочный:** каждые 25-50 мин (помодоро) -- «Я на задаче?»
- **Среднесрочный:** начало/конец дня -- «Двигаюсь ли к результату недели?»
- **Долгосрочный:** еженедельно (стратегирование) -- «Это приближает к целям?»

# Формализация: Цикл адекватного управления (Adequate Control Cycle)

## 1. Определение

Триадный цикл **Восприятие → Мышление/Интерпретация → Действие → (обратно к Восприятию)**, который работает только как замкнутая система, а не по частям или попарно.

Адекватность мышления, эмерджентная связность восприятия-мышления-действия — работает только в цикле.

## 2. Структура цикла

```
┌─────────────────────────────────────────────┐
│                                             │
│   ┌─────────────┐      ┌─────────────┐    │
│   │ Восприятие  │─────▶│  Мышление/  │    │
│   │             │      │Интерпретация│    │
│   └─────────────┘      └─────────────┘    │
│         ▲                      │           │
│         │                      │           │
│         │                      ▼           │
│         │              ┌─────────────┐    │
│         └──────────────│  Действие   │    │
│                        └─────────────┘    │
│                                             │
└─────────────────────────────────────────────┘
```

## 3. Три компонента цикла

| Компонент | Функция | Что даёт | Что происходит при выпадении |
|-----------|---------|----------|------------------------------|
| Восприятие | Считывание актуальной ситуации | Данные о реальности | Действие вслепую, галлюцинации |
| Мышление/Интерпретация | Осмысление через модель | Понимание, смысл | Бессмысленное восприятие и действие |
| Действие | Изменение ситуации | Воздействие на реальность | Пассивность, «паралич анализом» |

## 4. Ключевое свойство: только в цикле

### Не работают пары

| Пара | Почему не работает | Результат |
|------|-------------------|-----------|
| Восприятие + Мышление | Нет обратной связи от действия | Теоретизирование без проверки реальностью |
| Мышление + Действие | Нет считывания актуальной ситуации | Действие по устаревшей или ложной модели |
| Восприятие + Действие | Нет интерпретации | Рефлекторное реагирование без понимания |

### Не работают одиночные компоненты

| Компонент | Без цикла | Результат |
|-----------|-----------|-----------|
| Только восприятие | Нет осмысления и действия | Пассивная регистрация без понимания и влияния |
| Только мышление | Нет данных и проверки | Схоластика, отрыв от реальности |
| Только действие | Нет считывания и осмысления | Хаотическая активность |

## 5. Применение к телесной практике

### Восприятие
- Проприоцепция: считывание положения тела, напряжений, дискомфорта
- Интероцепция: считывание внутренних состояний, усталости, энергии

### Мышление/Интерпретация
- Распознавание паттернов напряжения
- Связь ощущений с моделью регуляции (например, иерархия регуляции)
- Формулирование гипотез о причинах состояния

### Действие
- Произвольная регуляция: смягчение, перераспределение усилия
- Изменение позы, дыхания, фокуса внимания
- Проверка гипотезы через изменение

### Обратная связь
- Восприятие результата действия
- Коррекция модели на основе результата
- Новый цикл регуляции

## 6. Применение к когнитивной деятельности

### Восприятие ситуации
- Считывание контекста задачи
- Распознавание ограничений и возможностей
- Обнаружение неявных требований

### Интерпретация через модель
- Применение релевантной ментальной модели (FPF, DDD, и т.д.)
- Формулирование гипотезы о решении
- Определение критериев успеха

### Действие
- Реализация решения
- Фиксация результата (код, текст, прототип)
- Измерение соответствия критериям

### Обратная связь
- Восприятие реального результата (тесты, отзывы, метрики)
- Коррекция модели при расхождении
- Новый цикл разработки

## 7. Критическое свойство: непрерывность

- Цикл должен быть **замкнутым**: каждый компонент получает вход от предыдущего
- Цикл должен быть **непрерывным**: нельзя «остановиться» на одном компоненте
- Цикл должен быть **быстрым**: чем короче цикл, тем выше адекватность управления

## 8. Практическое значение

### Для создателя
- Адекватность управления = способность цикла работать непрерывно
- Чем быстрее цикл, тем точнее регуляция
- Остановка цикла на любом компоненте = потеря адекватности

### Для развития
- Развитие = ускорение и углубление цикла
- Новичок: цикл медленный, осознанный
- Мастер: цикл быстрый, частично автоматический, но доступный произвольному вниманию

### Для диагностики
- Где застревает цикл?
- Восприятие: не считывается ситуация?
- Мышление: нет адекватной модели?
- Действие: не реализуется решение?

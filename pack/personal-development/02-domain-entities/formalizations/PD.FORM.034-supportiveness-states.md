# Формализация: Состояния опорности (Supportiveness States)

## 1. Определение

Пять дискретных состояний взаимодействия центра тяжести тела с площадью опоры, каждое из которых запускает специфические непроизвольные фоновые реакции.

Постуральная компетентность определяет утомляемость, комфорт за рабочим столом, способность поддерживать длительные рабочие сессии.

## 2. Пять состояний опорности

| Состояние | Положение центра тяжести | Фоновая реакция | Характер усилия |
|-----------|-------------------------|-----------------|-----------------|
| 1. Стояние | Внутри площади опоры | Минимальная тоническая активность | Стабилизация |
| 2. Баланс | На границе площади опоры | Микрокоррекции | Тонкая настройка |
| 3. Развитие падения | Вне площади опоры | Поиск новой опоры | Активное перемещение |
| 4. Торможение падения | Децентрализация | Экстренное торможение | Резкое противодействие |
| 5. Приземление | Установление новой опоры | Амортизация | Распределение нагрузки |

## 3. Описание состояний

### Состояние 1: Стояние
- Центр тяжести находится внутри площади опоры
- Тело в относительном покое
- Минимальные фоновые реакции для поддержания вертикали
- Наименее энергозатратное состояние

### Состояние 2: Баланс
- Центр тяжести на границе площади опоры
- Непрерывные микрокоррекции
- Высокая чувствительность к малейшим смещениям
- Требует развитой проприоцепции

### Состояние 3: Развитие падения
- Центр тяжести вышел за пределы опоры
- Активный поиск новой точки опоры
- Контролируемое перемещение
- Нормальная часть движения (ходьба, бег)

### Состояние 4: Торможение падения
- Неконтролируемое падение
- Экстренная децентрализация
- Резкие защитные реакции
- Высокая вероятность травмы при низкой компетентности

### Состояние 5: Приземление
- Установление новой площади опоры
- Амортизация ударной нагрузки
- Распределение усилия по всему телу
- Переход к новому циклу опорности

## 4. Практическое значение

### Для работы за столом
- Состояние 1 (стояние) должно быть доминирующим
- Плохая осанка = постоянное состояние 2 (баланс) → быстрая утомляемость
- Неправильная высота стула = регулярные микропадения → фоновое напряжение

### Для физической активности
- Компетентность в состояниях 3-5 определяет травмобезопасность
- Развитие падения (3) — ключевой навык для спорта и танца
- Торможение (4) и приземление (5) — критически важны для предотвращения травм

### Для долгосрочного здоровья
- Нераспознанные состояния 2-4 → хроническое фоновое напряжение
- Неосознаваемые компенсации → накопление микротравм
- Развитие чувствительности к состояниям → снижение энергозатрат на 30-50%

# Состояния созидателя

## 1. Концепция

**Связь с характеристиками:**

Состояние системы — это совокупность текущих значений показателей её характеристик.

```
Характеристики → Показатели → Значения → Состояние
```

## 2. Категории состояний

### 2.1. Продуктивное состояние

**Целевое операционное состояние** для работы.

| Параметр | Значение |
|----------|----------|
| Шкала | 0-5 баллов |
| Цель | Достижение и удержание уровня 4+ |
| Факторы | Сон, питание, режим, фокус |

### 2.2. Эмоциональные состояния

Эмоции как состояния:

| Состояние | Характер | Влияние на работу |
|-----------|----------|-------------------|
| Радость | Позитивное | Повышает креативность |
| Тревога | Негативное | Снижает фокус |
| Вдохновение | Позитивное | Повышает мотивацию |
| Раздражение | Негативное | Снижает качество решений |
| Апатия | Нейтральное-негативное | Снижает энергию |

### 2.3. Ролевые состояния

Какую роль человек исполняет сейчас:

| Состояние | Признаки |
|-----------|----------|
| В роли Ученика | Освоение нового, выполнение заданий |
| В роли Профессионала | Производство рабочих продуктов |
| В роли Исследователя | Генерация гипотез, работа с новизной |
| Вне роли | Отдых, переключение, быт |

### 2.4. Состояния неудовлетворённости

Мотивационные дефициты — ощущение нехватки, запускающее изменения:

| Состояние | Триггер | Результат |
|-----------|---------|-----------|
| Нехватка знаний | Непонимание | Обучение |
| Нехватка ресурсов | Ограничения | Поиск ресурсов |
| Нехватка признания | Игнорирование | Демонстрация результатов |
| Нехватка смысла | Пустота | Рефлексия, пересмотр целей |

### 2.5. Состояния намерения

Готовность к действию — состояние «собираюсь сделать»:

| Состояние | Уровень готовности |
|-----------|-------------------|
| Размышление | Низкий (0-20%) |
| Планирование | Средний (20-50%) |
| Готовность | Высокий (50-80%) |
| Действие | Максимальный (80-100%) |

### 2.6. Физические состояния

Самочувствие:

| Состояние | Описание |
|-----------|----------|
| Бодрость | Высокий уровень энергии |
| Усталость | Сниженный уровень энергии |
| Голод | Потребность в еде |
| Боль | Сигнал о проблеме |
| Сонливость | Потребность в отдыхе |

### 2.7. Социальные состояния

Статус и включённость в сообщество:

| Состояние | Описание |
|-----------|----------|
| Включённость | Активное участие в сообществе |
| Изоляция | Отстранённость от социума |
| Лидерство | Ведение за собой |
| Следование | Исполнение указаний |

### 2.8. Когнитивные состояния

Состояния мышления:

| Состояние | Описание | Влияние |
|-----------|----------|---------|
| Фокус | Высокая концентрация | +++ продуктивность |
| Рассеянность | Низкая концентрация | -- продуктивность |
| Перегрузка | Избыток информации | -- качество |
| Поток | Оптимальное состояние | +++ все показатели |
| Ступор | Блокировка мышления | --- все показатели |

## 3. Использование

### 3.1. Для самодиагностики

Оценка текущего состояния для принятия решений о деятельности.

### 3.2. Для наставничества

Понимание состояния ученика для адаптации подхода.

### 3.3. Для цифрового двойника

Отслеживание динамики состояний.

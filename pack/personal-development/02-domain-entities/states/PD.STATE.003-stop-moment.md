# Состояние: Стоп-момент (Stop Moment)

## 1. Определение

Стоп-момент -- микро-состояние осознанной паузы, в котором агент прерывает текущую деятельность для проверки выровненности: правильная ли задача? правильная ли роль? правильный ли приоритет? Стоп-момент -- фундаментальная единица собранности и основной инструмент против разобранности (PD.STATE.002).

## 2. Содержание стоп-момента

| Проверка | Вопрос | Что делать при несовпадении |
|----------|--------|----------------------------|
| **Задача** | То ли я сейчас делаю? | Вернуться к плану; переключиться на приоритетную задачу |
| **Роль** | В правильной ли я роли? | Осознанно выйти из текущей роли и войти в нужную |
| **Приоритет** | Это важное или срочное? | Различить (PD.FORM.024, действие #1); выбрать важное |
| **Масштаб** | Как это связано с целью недели / стратегией? | Восстановить связь с более крупным масштабом (PD.FORM.026) |
| **Состояние** | Как я себя чувствую? Сфокусирован? | При усталости -- отдых; при рассеянности -- ритуал входа |

## 3. Характеристики стоп-момента

| Параметр | Значение |
|----------|----------|
| **Длительность** | 5-30 секунд (микро-пауза, не полноценная рефлексия) |
| **Частота** | Чем чаще, тем лучше; минимум -- при переключении задач |
| **Триггер** | Таймер (помодоро), завершение задачи, ощущение потери фокуса, внешнее прерывание |
| **Результат** | Осознанное решение: продолжить / переключиться / отдохнуть |

## 4. Стоп-момент как индикатор собранности

```
Частота стоп-моментов  -->  Уровень собранности

Редко (0-1 в день)     -->  Разобранность (PD.STATE.002)
Иногда (3-5 в день)    -->  Базовый уровень
Регулярно (10+ в день) -->  Высокая собранность
Автоматически          -->  Привычка (неосознанная компетентность, PD.FORM.023)
```

Частота стоп-моментов -- наблюдаемый показатель уровня собранности. По мере практики стоп-моменты становятся автоматическими (переход в неосознанную компетентность).

## 5. Стоп-момент в ролевом мастерстве

В контексте ролевого мастерства (PD.FORM.031) стоп-момент выполняет функцию:
- **При входе в роль:** осознанная пауза перед началом -- «Какую роль я принимаю?»
- **При выходе из роли:** фиксация результата -- «Что я сделал? Что осталось?»
- **При переключении:** завершение текущей роли до начала следующей -- предотвращение смешения ролей

## 6. Типичные триггеры

| Триггер | Когда использовать |
|---------|-------------------|
| Таймер (помодоро) | Каждые 25-50 минут |
| Завершение задачи | Перед переходом к следующей |
| Входящее сообщение | Перед тем как ответить -- «Это важно сейчас?» |
| Ощущение дискомфорта | Тревога, скука, раздражение -- сигнал для паузы |
| Смена контекста | Переезд, смена места, конец встречи |

# Состояние: Перевозбуждение (Overexertion)

## 1. Определение

Перевозбуждение — состояние, при котором усилие превышает порог регуляции нервной системы. Нервная система не выдаёт расслабление после усилия, и каждое последующее усилие наслаивается на остаточное напряжение.

Создаёт порочный круг: больше напряжения → больше усталости → боль к вечеру → компенсаторное напряжение.

## 2. Характеристики состояния

| Параметр | Значение |
|----------|----------|
| Остаточное напряжение | Присутствует после усилия (нет расслабления) |
| Ответ НС | Нет релаксационного ответа (застревание в симпатической активации) |
| Накопление усталости | Экспоненциальное (каждое усилие усиливает предыдущее) |
| Боль | Появляется к вечеру, нарастает в течение дня |
| Скованность | Утренняя и вечерняя, не проходит после отдыха |

## 3. Обнаружение и триггеры

**Признаки перевозбуждения:**
1. Остаточное напряжение после окончания усилия (мышца не расслабляется)
2. Нет релаксационного ответа (не приходит «волна расслабления» после нагрузки)
3. Эскалация усталости в течение дня (утром терпимо, к вечеру невыносимо)
4. Боль или скованность к вечеру (плечи, шея, поясница)
5. Утренняя скованность (не проходит после разминки)

**Триггеры (что вызывает перевозбуждение):**
- Усилие выше порога регуляции (PD.CHR.004, уровень владения недостаточен)
- Слишком длительное удержание напряжения без пауз
- Работа в слепой зоне (PD.STATE.005) — нет обратной связи, усилие неконтролируемо растёт
- Игнорирование сигналов усталости (продолжение работы «через не могу»)

## 4. Влияние на продуктивное состояние

Перевозбуждение **блокирует** продуктивное состояние (PD.STATE.001):

| Аспект продуктивности | Влияние перевозбуждения |
|----------------------|-------------------------|
| Концентрация | Снижается (телесный дискомфорт отвлекает) |
| Работоспособность | Падает экспоненциально (усталость накапливается) |
| Качество работы | Ухудшается (ошибки, медленная реакция) |
| Восстановление | Не происходит (нет расслабления даже после отдыха) |

**Порочный круг:**
Перевозбуждение → боль → компенсация позой → дополнительное напряжение → ещё больше боли → выход из продуктивного состояния.

## 5. Стратегия выхода

**Немедленные действия:**
1. **Уменьшить усилие ниже порога регуляции** (исследовательский шаг, PD.D.082)
2. **Дождаться релаксационного ответа** (не продолжать, пока не придёт расслабление)
3. **Увеличить длительность удержаний** (дать НС время на адаптацию, PD.D.081)
4. **Увеличить длительность пауз** (2-3 минуты между подходами вместо 30 секунд)

**Долгосрочная профилактика:**
1. Развитие телесной саморегуляции (PD.CHR.004) — освоение более тонкого шага усилия
2. Устранение слепых зон (PD.STATE.005) — через протокол чувствительности (PD.D.083)
3. Соблюдение правила «меньше = больше» (80% от максимального усилия)
4. Ведение дневника нагрузок (PD.FORM.033) — отслеживание порога регуляции

## 6. Связанные документы

- PD.D.081 — Протокол удержания (безопасная работа с длительными усилиями)
- PD.CHR.004 — Телесная саморегуляция (развитие способности регулировать усилие)
- PD.STATE.001 — Продуктивное состояние (блокируется перевозбуждением)
- PD.D.082 — Исследовательский шаг (поиск меньшего усилия)
- PD.STATE.005 — Слепая зона (одна из причин перевозбуждения)

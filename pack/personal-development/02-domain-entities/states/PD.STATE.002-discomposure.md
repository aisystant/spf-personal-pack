# Состояние: Разобранность (Discomposure)

## 1. Определение

Разобранность (discomposure) -- самоподдерживающееся анти-состояние, качественно отличное от простого отвлечения. Характеризуется петлёй положительной обратной связи: потеря фокуса приводит к ошибкам, ошибки -- к потере ресурсов (время, энергия, репутация), потеря ресурсов -- к ещё большей потере фокуса. Без сознательного вмешательства состояние углубляется.

## 2. Механизм самоподдержания

```
Потеря фокуса
     |
     v
Ошибки / некачественные решения
     |
     v
Потеря ресурсов (время, энергия, доверие)
     |
     v
Стресс / фрустрация
     |
     v
Ещё большая потеря фокуса  -->  цикл повторяется
```

Петля самоподдерживающаяся: каждый виток усиливает следующий. В отличие от простого отвлечения (PD.FORM.025), разобранность -- системное состояние, затрагивающее все сферы деятельности.

## 3. Отличие от простого отвлечения

| Аспект | Простое отвлечение | Разобранность |
|--------|-------------------|---------------|
| **Масштаб** | Одна задача / один момент | Все сферы деятельности |
| **Длительность** | Минуты | Дни, недели, месяцы |
| **Динамика** | Линейная (отвлёкся -- вернулся) | Самоподдерживающаяся петля (углубляется) |
| **Выход** | Стоп-момент достаточен | Требуется осознанная установка практик собранности |
| **Симптомы** | Потерял нить задачи | Не может назвать приоритеты; хаотичная деятельность; реактивный режим |

## 4. Симптомы разобранности

| Симптом | Проявление |
|---------|-----------|
| Неспособность назвать приоритеты | «Не знаю, что сейчас важнее» |
| Реактивный режим | Весь день отвечает на входящие, ничего стратегического не делает |
| Потеря временных масштабов | Не помнит план недели; не связывает задачу с целью |
| Хаотичное переключение | Начинает много, завершает мало |
| Эмоциональная нестабильность | Раздражительность, тревога, апатия без явной причины |
| Прокрастинация | Откладывание важного в пользу лёгкого/срочного |

## 5. Выход из разобранности

Выход требует осознанного усилия по установке практик собранности (PD.FORM.024):

| Шаг | Действие | Цель |
|-----|----------|------|
| 1 | **Остановка** (стоп-момент) | Прервать петлю; осознать состояние |
| 2 | **Диагностика** | Назвать: что потеряно (масштаб, приоритет, роль)? |
| 3 | **Упрощение** | Убрать лишнее; выбрать одну задачу |
| 4 | **Ритуал входа** | Войти в рабочий режим через защищённое время + ритуал |
| 5 | **Регулярность** | Установить регулярные стоп-моменты для предотвращения рецидива |

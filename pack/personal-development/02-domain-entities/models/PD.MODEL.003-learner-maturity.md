# Модель: Ступени зрелости ученика (Learner Maturity Levels)

## 1. Определение

**Ступени зрелости ученика** — квалификационная модель, описывающая прогресс в освоении роли Ученика от случайного обучения к проактивному саморазвитию.

## 2. Пять ступеней

### Ступень 1: Случайный ученик

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Учусь, когда заставили или когда настроение |
| **Триггер обучения** | Внешнее давление, эмоциональный импульс |
| **Систематичность** | Отсутствует |
| **Перенос** | Случайный |

### Ступень 2: Практикующий ученик

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Учусь по настроению и часто без плана |
| **Триггер обучения** | Стимул (курс начался, дедлайн) |
| **Систематичность** | Эпизодическая |
| **Перенос** | Частичный |

### Ступень 3: Систематический ученик

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Учусь "по процессу" почти каждый день |
| **Триггер обучения** | Ритм, привычка |
| **Систематичность** | Есть, но неполная (<10 ч/нед) |
| **Перенос** | Регулярный, но без глубокого переосмысления |

### Ступень 4: Дисциплинированный ученик

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Строю личную траекторию развития |
| **Триггер обучения** | Внутренняя цель |
| **Систематичность** | Полная (≥10 ч/нед) |
| **Перенос** | Глубокий, с осмыслением |

### Ступень 5: Проактивный ученик

| Параметр | Значение |
|----------|----------|
| **Характеристика** | Строю траекторию + понимаю роль культуры саморазвития |
| **Триггер обучения** | Миссия, вклад в мир |
| **Демонстрация** | Демонстрирует культуру в поведении и проектах |
| **Участие** | Активно участвует в сообществе, берёт инициативу |
| **Передача** | Передаёт культуру саморазвития другим |
| **Мастерство** | Максимальные показатели по всем практикам |

## 3. Критерии перехода

```
Случайный → Практикующий
    │ Регулярность хотя бы эпизодическая
    ↓
Практикующий → Систематический
    │ Ежедневность практики
    ↓
Систематический → Дисциплинированный
    │ 10+ ч/нед + личная траектория
    ↓
Дисциплинированный → Проактивный
    │ Культура + передача + сообщество
```

## 4. Диагностические вопросы

| Ступень | Ключевой вопрос |
|---------|-----------------|
| 1→2 | Есть ли хотя бы эпизодическая практика обучения? |
| 2→3 | Учусь ли я почти каждый день? |
| 3→4 | Есть ли личная траектория и 10+ ч/нед? |
| 4→5 | Передаю ли я культуру другим и участвую ли в сообществе? |

## 5. Связь с программами

| Ступень | Минимум для программы |
|---------|----------------------|
| 3 | Вход в программу Личного развития |
| 4 | Завершение программы Личного развития |
| 5 | Вход в программу Рабочего развития |

# Модель: 6 направлений саморазвития

## 1. Назначение

Модель описывает шесть взаимосвязанных направлений личного развития, образующих единый поток:

```
Организм/ресурсы (6) → Внимание (2,3) → Знания/Экзокортекс (4)
    → Продукты и Репутация (5) → Коррекция мировоззрения (1)
```

## 2. Шаблон описания направления

Каждое направление описывается по структуре:
1. **Кратко о чём** — смысловая зона развития
2. **Артефакты** — что остаётся после работы
3. **Объекты внимания** — конкретные аспекты
4. **Методы и микропрактики** — что поддерживает поток
5. **Типовые узкие места** — ловушки и ошибки
6. **Метрики** — показатели прогресса
7. **Проверка переноса** — что поменялось за 7–14 дней

## 3. Шесть направлений

### 3.1. Картины мира и мировоззрение

| Параметр | Значение |
|----------|----------|
| **Кратко** | Смыслы, цели, мемы-скрипты, агентность, калибр личности |
| **Артефакты** | Карта мемов и антитезисов; личная формула смысла; журнал поворотов |
| **Объекты внимания** | Смыслы, критерии, когнитивные искажения, самоэффективность |
| **Методы** | 1 мем → 1 антитезис → 1 микро-эксперимент/нед; мышление письмом |
| **Узкие места** | Сертификаты вместо продуктов; фанатизм терминов |
| **Метрики** | Кол-во мемов/антитезисов; частота поворотов |

### 3.2. Навыки и мастерство (ролевая траектория)

| Параметр | Значение |
|----------|----------|
| **Кратко** | Дисциплина ученика → понятийная работа → качество профессионала → новизна исследователя → ясность просветителя |
| **Артефакты** | Недельный контракт; портфель кейсов; карта ворот между ролями |
| **Объекты внимания** | Рабочие продукты; ритм слотов; понятийная сетка; публичные инкременты |
| **Методы** | 3×25–50 мин фокус-слотов/день; чек-листы качества; задача→артефакт→демо |
| **Узкие места** | Перфекционизм; учёба без переноса; многозадачность |
| **Метрики** | Продуктов/нед; доля задач с переносом; доля встреч с протоколом |

### 3.3. Ограничения и узкие горлышки

| Параметр | Значение |
|----------|----------|
| **Кратко** | Состояния Хаос→Тупик→Поворот→Развитие; поведенческие петли; зависимости |
| **Артефакты** | Журнал состояний и триггеров; теплокарта срывов; план анти-петель |
| **Объекты внимания** | Стабильно плохое состояние; триггеры срывов; геройские рывки |
| **Методы** | Стоп-момент; двухконтурный отдых; декомпозиция микро-шаг |
| **Узкие места** | Ночь вместо режима; ещё одно письмо/лента/сериал |
| **Метрики** | Частота срывов; длина поворота; серия продуктивных дней |

### 3.4. Экзокортекс и ИИ (киберличность)

| Параметр | Значение |
|----------|----------|
| **Кратко** | Внешняя память и вычислительный слой: инбокс→заметка→черновик→заготовка→продукт |
| **Артефакты** | Журнал промптов; тест-наборы; карты понятий; реестр артефактов |
| **Объекты внимания** | Структура БЗ; логирование ИИ; приватность; воспроизводимость |
| **Методы** | Факт-чек-3; тест-подзадача без ИИ; ревизия экзокортекса 2×нед |
| **Узкие места** | Свалка ссылок; хрупкие ответы; нечёткая граница авторства |
| **Метрики** | Доля завершённых цепочек; время поиска; экспорт из БЗ в проекты |

### 3.5. Окружение, культура и рабочие системы

| Параметр | Значение |
|----------|----------|
| **Кратко** | Физическая и социально-операционная среда; репутация и эпистемический статус |
| **Артефакты** | Операционная мантра; протоколы встреч; витрина инкрементов; публичные кейсы |
| **Объекты внимания** | Защита слотов; культура промежуточек; каналы сообществ |
| **Методы** | 1 продукт недели/чел; ревью по критериям; нулевой стол |
| **Узкие места** | Видимость без результата; болталки без артефактов |
| **Метрики** | Регулярность ревью; доля встреч с протоколом; рост тёплых входящих |

### 3.6. Организм и личные ресурсы

| Параметр | Значение |
|----------|----------|
| **Кратко** | Биофизическая основа: сон, питание, движение, стресс-менеджмент; впечатления как допинг мотивации |
| **Артефакты** | 7-дневный график сна/движения; дневник энергии; журнал впечатлений |
| **Объекты внимания** | Продуктивное состояние; сон/фазы; ежедневное движение; бюджет ресурсов |
| **Методы** | Режим сна; движение каждый день; гигиена уведомлений; микро-радости |
| **Узкие места** | Недосып; компенсация усталости быстрыми удовольствиями |
| **Метрики** | ≥5 ночей по 7+ ч; доля дней с активностью; доля фокус-часов |

## 4. Проверка переноса

Для каждого направления: **Что из этого блока поменяло план слотов и выбор методов на неделе?**

# Work Products: Работы ученика (Learner Works)

## 0. Назначение

Этот документ описывает **работы (work products) роли Ученика** — конкретные результаты деятельности, которые подтверждают освоение роли.

**FPF A.7**: Работа ≠ Роль ≠ Метод
- **Роль** → [PD.ROLE.001](../02-domain-entities/roles/PD.ROLE.001-learner.md)
- **Метод** → [PD.METHOD.002](../03-methods/PD.METHOD.002-learner-method.md)
- **Работа** (этот документ) — что Ученик производит

---

## 1. Определение (FPF A.15.1)

**Работы ученика** — записи о выполненных действиях и созданных артефактах, которые подтверждают прохождение учебного процесса.

```
Роль Ученик + Метод обучения → Работы ученика
                                    ↓
                              Evidence для перехода
```

---

## 2. Обязательные артефакты

### 2.1. Недельные контракты

| Параметр | Значение |
|----------|----------|
| **Формат** | Markdown-файл по шаблону |
| **Частота** | Еженедельно |
| **Минимум** | 8 контрактов для перехода |

### 2.2. Заметки в экзокортексе

| Параметр | Значение |
|----------|----------|
| **Формат** | Атомарные заметки (Zettelkasten) |
| **Минимум** | 50 заметок |
| **Связность** | ≥20% заметок связаны между собой |

### 2.3. Еженедельные рефлексии

| Параметр | Значение |
|----------|----------|
| **Формат** | Текст 300-500 слов |
| **Частота** | Еженедельно |
| **Содержание** | Что изучил, что понял, что сложно |

### 2.4. Журнал мировоззренческих поворотов

| Параметр | Значение |
|----------|----------|
| **Формат** | Записи о "ага-моментах" |
| **Минимум** | 3 записи |
| **Содержание** | Что изменилось в понимании мира |

---

## 3. Опциональные артефакты

| Артефакт | Описание | Бонус |
|----------|----------|-------|
| **Конспект курса** | Структурированное изложение | +1 к оценке |
| **Учебный проект** | Практическое применение | +2 к оценке |
| **Пост/статья** | Публичное изложение | +1 к оценке |
| **Помощь другому ученику** | Peer-support | +1 к оценке |

---

## 4. Оценка работ

### 4.1. Критерии оценки

| Критерий | Вес | Описание |
|----------|-----|----------|
| **Регулярность** | 30% | Выполнение слотов, контракты |
| **Качество заметок** | 30% | Структура, связность, глубина |
| **Рефлексия** | 20% | Осмысленность, честность |
| **Прогресс** | 20% | Рост от начала к концу |

### 4.2. Шкала оценки

| Оценка | Критерии |
|--------|----------|
| **Отлично** | Все артефакты + опциональные, высокое качество |
| **Хорошо** | Все артефакты, хорошее качество |
| **Удовлетворительно** | Минимум артефактов, базовое качество |
| **Недостаточно** | Не все артефакты или низкое качество |

## 5. Портфолио ученика

### 5.1. Структура портфолио

```
portfolio-ученик/
├── contracts/              # Недельные контракты
│   ├── week-01.md
│   ├── week-02.md
│   └── ...
├── reflections/            # Еженедельные рефлексии
│   ├── 2025-01-week1.md
│   └── ...
├── worldview-turns/        # Мировоззренческие повороты
│   └── turns.md
├── exocortex/              # Экзокортекс (или ссылка)
│   └── (заметки)
└── README.md               # Обзор портфолио
```

### 5.2. Пример README

```markdown
# Портфолио ученика: [Имя]

## Статус: Ученик (2 месяца)

## Артефакты:
- Недельных контрактов: 8/8 ✓
- Заметок в экзокортексе: 67 ✓
- Рефлексий: 8/8 ✓
- Мировоззренческих поворотов: 4 ✓

## Ключевые достижения:
- Освоил базу системного мышления
- Установил устойчивый ритм 10ч/нед
- Запустил экзокортекс в Obsidian

## Готовность к переходу: ДА
```

## 6. Валидация перехода к Интеллектуалу

### 6.1. Чек-лист готовности

- [ ] ≥8 недельных контрактов (80%+ выполнение)
- [ ] ≥50 заметок в экзокортексе
- [ ] ≥8 еженедельных рефлексий
- [ ] ≥3 мировоззренческих поворота
- [ ] Устойчивый ритм ≥10ч/нед саморазвития
- [ ] Оценка наставника ≥ "Хорошо"

# Метод: Обучение ученика (Learner Method)

## 0. Назначение

Этот документ описывает **метод обучения для роли Ученика** — как именно ученик осваивает материал и развивается.

## 1. Определение (FPF A.3.1)

**Метод обучения ученика** — набор практик и процедур, которые позволяют систематически осваивать знания и формировать навыки саморазвития.

---

## 2. Ключевые принципы

| Принцип | Описание |
|---------|----------|
| **Регулярность** | Ежедневные/еженедельные слоты |
| **Рефлексия** | Осмысление опыта после каждого слота |
| **Структурирование** | Ведение экзокортекса |
| **Контрактирование** | Письменные обязательства |
| **Итерации** | Цикл "план → действие → рефлексия" |

---

## 3. Процедура метода

### 3.1. Недельный цикл

```
Понедельник: Планирование недели, недельный контракт
Вт-Пт: Учебные слоты (по 1-2 часа)
Суббота: Еженедельная рефлексия
Воскресенье: Отдых, чтение по интересам
```

### 3.2. Структура учебного слота

```
0-5 мин:   Настройка (убрать отвлекающие)
5-15 мин:  Повторение предыдущего материала
15-55 мин: Новый материал (читать/смотреть/делать)
55-60 мин: Заметка в экзокортекс + микро-рефлексия
```

### 3.3. Недельный контракт (шаблон)

```markdown
# Недельный контракт ученика

Неделя: [дата начала - дата конца]

## Обязательства:
- [ ] Провести N учебных слотов
- [ ] Написать M заметок в экзокортекс
- [ ] Выполнить практическое задание по теме X
- [ ] Провести еженедельную рефлексию

## Награда за выполнение:
[что получу, если выполню]

## Последствия невыполнения:
[что произойдёт, если не выполню]

Подпись: _______________
```

## 4. Инструменты

| Инструмент | Назначение | Примеры |
|------------|------------|---------|
| **Календарь** | Планирование слотов | Google Calendar, Notion |
| **Экзокортекс** | Структурирование знаний | Obsidian, Notion, Roam |
| **Трекер привычек** | Отслеживание регулярности | Habitica, Notion |
| **Дневник рефлексий** | Фиксация осмыслений | Markdown-файл |

**Важно**: Инструмент ≠ Метод (FPF различение). Метод можно выполнять разными инструментами.

---

## 5. Метрики успешности

| Метрика | Целевое значение | Как измерять |
|---------|------------------|--------------|
| % выполненных слотов | ≥80% | Трекер привычек |
| Заметок в экзокортексе | ≥5/неделю | Подсчёт |
| Еженедельные рефлексии | 100% | Чек-лист |
| Выполнение контракта | ≥80% | Самооценка |

---

## 6. Типичные ошибки (Failure Modes)

| Ошибка | Последствие | Решение |
|--------|-------------|---------|
| Нерегулярность | Потеря навыка, срывы | Жёсткие слоты в календаре |
| Отсутствие рефлексии | Знания не усваиваются | Обязательная заметка после слота |
| Слишком амбициозные планы | Выгорание, срыв | Начать с минимума (2 слота/нед) |
| Нет экзокортекса | Знания теряются | Завести простую систему заметок |
| Путаница инструмент=метод | Смена инструмента = срыв | Фокус на методе, не на инструменте |
